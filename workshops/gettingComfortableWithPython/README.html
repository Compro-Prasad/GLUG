Formatted(f) strings

Also known as string interpolation

Works in Python 3.6 and above

Normal string

def get_all_results(table_name):
     return run_query(
          "SELECT * from " + table_name + ";"
     )

Using f string

def get_all_results(table_name):
     return run_query(
          f"SELECT * from {table_name};"
     )

Exception(Error) handling

Sample Code

student = {
    "name": "Ram"
}
print(student["mother"])

Find the error.

Error

Traceback (most recent call last):
File "<stdin>", line 4, in <module>
KeyError: 'mother'

Here type of the error is KeyError. All errors have this information at the end of error output.

Catching errors

student = {
    "name": "Ram"
}
try:
    print(student["mother"])
except KeyError as error:
    print(f"Key not found {error}")

Key not found 'mother'

Catching multiple errors?

student = {
    "name": "Ram",
    "roll": 1
}
keys = ["name", "roll", "dob"]
for key in keys:
    print("Output:", student[key] + 1)

Catching multiple errors (Solution)

student = {
    "name": "Ram",
    "roll": 1
}
keys = ["name", "roll", "dob"]
for key in keys:
    try:
        print("Output:", student[key] + 1)
    except KeyError as error:
        print(f"Key not found {error}")
    except TypeError:
        print(f"Can't add 1 to '{key}'")
    except Exception as error:
        # Catches all other errors
        print(f"Key: '{key}', Error: {error}")

Constructor and Destructor

class Vehicle:
    """Vehicle docstring"""

    def __init__(self, name, max_speed):
        # constructor
        # Public members
        self.name = name
        self.max_speed = max_speed
        # Private members begin with _
        self._current_speed = 0
        print("Constructed")

    def __del__(self):
        # Destructor
        print("Destroyed")

Inheritance

class Car(Vehicle):
    """Car doc"""
    def __init__(self, name, max_speed):

        # Call base class constructor
        super().__init__(name, max_speed)

        # Create new members
        self.wheels = 4

Multiple inheritance (Simple way)

class FlyingCar(Car, Airplane):
    """Docstring for FlyingCar"""

    def __init__(self, name, max_speed):
        Car.__init__(self, name, max_speed)

Polymorphism

Square class

class Square:
    """Square doc"""
    def __init__(self, side):
        self.side = side
    def area(self):
        """Find the area"""
        return self.side * self.side

Circle class

class Circle:
    """Circle doc"""
    def __init__(self, radius):
        self.radius = radius
    def area(self):
        """Find the area"""
        return 3.1416 * self.radius * self.radius

Calculate area

print(Square(5).area())
print(Circle(5).area())

Defining __str__ and __repr__

class Vehicle:
    """Vehicle docstring"""
    def __init__(self, name, max_speed):
        self.name = name
        self.max_speed = max_speed
        self._current_speed = 0
    def __str__(self):
        # Helps in converting to string
        return f"{self.name}: {self.max_speed}"
    def __repr__(self):
        # Helps in seeing a better version
        return f"<{self}>"

Python Virtual Environment

Why is it used?

Isolate different versions of python and python packages.

How to setup using venv

python -m venv testenv

Creates a new directory testenv with the python version and builtin modules.

Loading the environment

source testenv/bin/activate  # *nix
testenv\Scripts\activate.bat  # Windows

This modifies environment variables like PATH which helps in prioritizing python from the virtual environment.

Installing packages

pip install requests
pip install -r requirements.txt

This will install all packages in the virtual environment when its activated.

Unloading the environment

deactivate

This reverts the changes done to the environment variables.
