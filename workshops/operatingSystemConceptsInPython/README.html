Defining functions

def isPrime(n):
    if n < 2:
        return False
    else if n == 2:
        return True
    else if n % 2 == 0:
        return False
    else:
        for i in range(3, int(n ** i) + 1, 2):
            if (n % i == 0):
                return False
        return True

Lambda functions

  • Also known as anonymous function
  • One line only
  • Used for defining in place functions

  • Syntax: 

    square = lambda x: x * x
square(4)

Merits of lambda

  • It gets the work done
  • If the function definitions are short

makeInc using lambda

def makeInc(m):
    return (lambda n: n + m)

Using makeInc

incBy2 = makeInc(2)
incBy3 = makeInc(3)
print(incBy2(7))
print(incBy3(32))

Output: 

9
35

What is missing?

  • Lambda is good in lisp like languages and JavaScript
  • But not particularly in python
  • Cannot write multiple statements

Classes and objects

Point to be noted

  • Python is not fully object oriented as it lacks encapsulation i.e. data hiding.
  • The concepts of object oriented programming is out of the scope of this class.

What is a class?

Basically it means defining a custom datatype from the predefined types and/or inheriting other types.

Initializing data members

Constructors are used for initializing data members of a class. 

class complex:
    def __init__(self, a: int, b: int):
        # Constructor
        self.real, self.imag = a, b

Class methods/member functions

class complex:
    def __init__(self, a: int, b: int):
        # Constructor
        self.real, self.imag = a, b
    def __add__(self, x: complex):
        y = complex(self.real + x.real,
                    self.imag + x.imag)
        return y
    def __str__(self):
        r = str(self.real)
        i = str(self.imag)
        return r + "+i" + i

Operating system

How do we do it?

Python provides an interface to the operating system through the os library of functions which contains several system calls to retrieve and manipulate different parts of operating system. 

import os

What is a process?

It is an instance of a computer program that is being executed. A process in execution requires a processing unit(like CPU) to run the instructions as specified in the program. 

import os
os.exec
os.fork
os.wait

What is forking?

  • A way of cloning processes.
  • Continues execution from the next line.
